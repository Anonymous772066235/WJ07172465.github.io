大作业要求

       《Web前端设计与开发》选修课旨在激发各位同学对于Web设计与开发的兴趣，所以不会安排考试环节，获取学分的方法是按照以下要求完成一次大作业，完成作业的同时其实是你创建自己第一个互联网网站的过程，全程无需任何费用，你需要付出的仅仅是时间和创造力。

1、作业内容：

       每位同学制作一个包含至少 5 个静态页面的个人网站

、作业要求：

    ① 从首页可以通过明显的方式（如超级链接）到达其它页面

    ② 网站通过 Coding.net 或 GitHub 平台提供的 Pages 服务（二选一，建议使用Coding .net）发布，将发布后的网址提交给我，发布的方法见后面说明

    ③ 内容不限、实现方法不限、实现水平不限，可以使用模版

    ④ 作业提交截止时间为 2019年12月10日 00:00:00 之前

如有同学有自己的个人网站，可以直接提交网址，但首页上必须有明显的个人信息（如姓名、班级等）标识以便认证。

3、提交方式：

将 Coding.net 或 GitHub Pages 服务的首页地址及其它相关信息提交到：

https://wj.qq.com/s2/4764321/a8fa

 

附件：

1、Coding .net 或 GitHub 平台 Pages 服务的使用可以参考《使用Coding Pages创建你的第一个网站.pdf》、《使用Github Pages创建你的第一个网站.pdf》和《GitHub Pages网站上传、提交及注意事项.pdf》

2、《电子商务2014《网页设计基础》设计作品展.pdf》是一个之前的学生作品集，可以参考一下他们的实现程度
